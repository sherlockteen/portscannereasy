# Simple python port scanner

## How to install:
```bash
git clone https://github.com/q1ncite/portscannereasy.git
cd  portscannereasy
python3 portscanner.py
```

## How to use:
Enter one or more ip addresses separated by commas, then write how many ports you want to scan
